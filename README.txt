== Description ==

Wise KPIs is a free plugin that displays almost 25 key performance indicators related to your WordPress website's performance. Whether you're a blogger or an eCommerce shop owner using WooCommerce, this plugin can help you easily track important metrics like comments, posts, orders, and users metrics. With Wise KPIs, you can make data-driven decisions to optimize your website for better results.

== KPIs ==

The plugin offers almost 25 KPIs for almost every section on your site including user engagement, content publishing, system insiders and much more.
Some of our KPIs
1. Revenue per customer
2. Number of revisions
3. Number of menu items
4. Average post length
5. Number of orders
6. Average comments per user
7. Number of authors
8. Coupons usage
9. Average Discounts
And much, much more...

= Features: =

1. Easily track important KPIs
2. Make data-driven decisions
3. Optimize your website for better results based on data
4. Generate KPIs data between date ranges
5. Change the comparison unit (yesterday, last week, last month etc...)
6. Auto refresh KPI pages

Free to download and use
* Users (1 KPIs)
* Posts (8 KPIs)
* Engagement (3 KPIs)
* System (4 KPIs)
* Woocommerce (6 KPIs)
* Simple installation process

== Installation ==

This section describes how to install the plugin and get it working.
e.ga
1. upload `wise-kpis.zip` to the `/wp-content/plugins` directory and unzip it.
2. Activate the plugin through 'Plugins' menu in wordpress.

You can now navigate to various KPI pages within the Wise KPIs section.
